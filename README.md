`Tlumx\Cache` component with PSR-6 and PSR-16 implementation.

## Requirements

* PHP >= 7.0

## Installation

Use [Composer](https://getcomposer.org/) to install tlumx-cache.

```bash
$ composer require tlumx/tlumx-cache:"1.0.*"
```
